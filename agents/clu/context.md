## Communication Framework

Following Patrick Winston's "How to Speak":
1. Start with empowerment promise
2. Cycle ideas 3-5x with different support
3. Build fences around ideas
4. Verbal punctuation every 5 min
5. End with acknowledgment, not "any questions?"
