# Glossary

## Terms

**Agentic Engineering** — Building with AI agents as collaborators, not just tools. Coined by Simon Willison.

**Dark Factory Pattern** — Automated code production where nobody writes or reviews code; AI does its own QA.

**Lethal Trifecta** — Three conditions for prompt injection vulnerability: private data, untrusted content, external communication.

**Prompt Injection** — Security vulnerability where untrusted content manipulates AI behavior. Coined by Simon Willison.

**T1/T2/A1/A2 Paradigms** — Adaptation patterns from Stanford/Harvard paper:
- A1: Tool execution signals agent improvement
- A2: Agent output signals improvement
- T1: Agent-agnostic tool adaptation
- T2: Agent-supervised tool adaptation

**Vibe Coding** — Informal coding style where developers describe intent and AI generates code. Coined by Andrej Karpathy.
