# Clu — Raw Capture Template

Use this template for daily raw captures.

```markdown
# Raw — YYYY-MM-DD

## Emails
- [summary of important emails]

## Calendar
- [upcoming events]

## News & Research
- [relevant updates]

## Observations
- [anything the agent noticed]

## Actions Taken
- [what the agent did today]
```

Place daily files in `agents/clu/raw/YYYY-MM-DD.md`.
