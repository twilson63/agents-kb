## Key Discovery: OpenRouter Image Generation

Critical parameter: `modalities: ["image", "text"]` must be included in request body. Without it, image generation fails silently or returns errors.

Available models:
- `gemini-nano` (cheapest) → google/gemini-2.5-flash-image
- `gemini-flash` → google/gemini-3.1-flash-image-preview
- `gpt-image` → openai/gpt-5.4-image-2
- `flux-pro` → black-forest-labs/flux.2-pro (only `["image"]`, may not work)

Script: `OPENROUTER_API_KEY="sk-or-..." OUTPUT_DIR="./output" node generate.mjs "<prompt>" gemini-nano`
