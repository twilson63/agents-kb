# AI Literacy with OpenClaw

A three-module, instructor-led course designed for non-technical users. Goal: help people develop the mental models and habits needed to work productively with AI agents as the world becomes more agent-driven.

**Format:** 3 × 50-minute modules, instructor-led
**Audience:** Non-technical users, new to AI agents
**Practice:** Each student has a HyperClaw practice pod

## Modules

| Module | Topic | Essays |
|--------|-------|--------|
| 1 | [Reverse Prompting](module-1-reverse-prompting.md) | 8 essays |
| 2 | [Skills & Memory](module-2-skills-and-memory.md) | 8 essays |
| 3 | [Multi-Agent & Delegation](module-3-multi-agent-delegation.md) | 7 essays |

## Production Flow

```
Essay outline → Draft essay → Humanizer pipeline → Slide deck → Video script → Video
```

Each essay is ~400-600 words, ~5-7 minutes presented.
All practice exercises use the student's HyperClaw practice pod.
